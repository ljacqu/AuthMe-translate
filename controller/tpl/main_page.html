Submitting Translations

You can submit translation changes in three different ways:
  1. Using this web interface, select the language and hit "Edit translation." This will create a personal copy of the translation you can edit and then share (instructions on the edit page).
  2. Submit a pull request with the changes to your messages_xx.yml file to the AuthMe dev repository.
  3. Upload your messages_xx.yml file (e.g. to hastebin) and open a new issue on the AuthMe issue tracker.
